# Real Time Weather Monitoring System
This project is a real-time weather monitoring system that continuously retrieves weather data from the OpenWeatherMap API for major metros in India. It logs the weather data, checks for alert thresholds, and visualizes temperature trends. The system is designed to be easy to configure and extend.

# Prerequisites
1. Python: Ensure latest version of python is downloaded.
2. Openweathermap API Key: Create an account on openweathermap and generate a free API key.
3. Git(optinal)
